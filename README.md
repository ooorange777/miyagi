# NoTomorrow Club

## About NoTomorrow Club

本博客由 Astro 框架搭建而成，仿照豆瓣个人主页的样式。将书影音记录替换成更开放的 letterboxd、goodreads、lastfm，广播替换为属于联邦宇宙的微博平台。

目前博客主体搭建完成，还有评论、标签、列表、相册等部分需要添加。
